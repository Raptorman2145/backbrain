Set Up Chat

This chat needs a free Firebase project to relay messages between devices — this site has no server of its own. Every message is encrypted in your browser before it's sent (see the Join screen), so Firebase only ever stores unreadable ciphertext.

One-time setup (a few minutes):
1. Go to console.firebase.google.com and create a project (no credit card needed).
2. Click the </> "add web app" icon, register an app, and copy the firebaseConfig object it shows you.
3. In the left menu: Build → Realtime Database → Create Database (any region; starting in locked mode is fine).
4. Open that database's Rules tab, replace the contents with the JSON below, then Publish.
5. Paste the config object above and continue.

Join Room

    
Enter the shared passphrase for your private room — treat it like a Discord invite link, anyone who knows it can join, read, and post. Messages are encrypted with a key derived from this passphrase, so even someone who found your Firebase project couldn't read them without it. This isn't per-person identity: anyone with the passphrase can post under any nickname.
